# 基于Gradio的简易车牌检测app

## 简介
- 使用Gradio搭建前端，方便简单
- 使用yolov5 6.0和LPRNet（旧版本的yolov5可能无法使用）

## 使用
### 克隆本仓库
```
git clone https://github.com/exhyy/gradio-yolov5-lprnet.git
```
### 安装依赖
```
pip install -r requirements.txt
```
### 运行
```
python main.py
```
